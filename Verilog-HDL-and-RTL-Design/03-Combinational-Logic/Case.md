# **case statement:**

- Imagine you are at a vending machine.

- You press a button based on what you want:

- If you press Button 1 You get 'Water'.

- If you press Button 2 You get 'Soda'.

- If you press Button 3 You get 'Juice'.

- If you press Any other button The machine says "Invalid Choice".

- In Verilog, a case statement does exactly this.

- It takes a control signal (like a selection switch) and matches it to a specific output path.

- Why use case instead of if-else?You can do everything with if-else, but if you have 4, 8, or 16 different choices, writing else if over and over makes your code messy and hard to read.

- A case statement keeps your code beautiful and easy for GitHub!
